# makemore

A bigram language model following Karpathy's Neural Networks Series

Trained on names, predicts the next character one at a time.

## What I learned
- Negative log-likelihood loss
- Model smoothing
- Count-based and neural net approaches converge to the same solution

## Files
- `makemore_simple.py` — count-based approach
- `makemore_nn.py` — neural network approach
